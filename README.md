Development Setup
--------------------------------------------------------------------------------

1. download [node](http://nodejs.org/)
2. install [bower](http://bower.io/)
3. run bower install (may need run as super user `sudo`)
4. run npm install (may need run as super user `sudo`)
5. run gulp serve
